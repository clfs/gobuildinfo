# gobuildinfo
Show `debug.BuildInfo` for Go binaries.

## Installation
Install `gobuildinfo`, or update it:

```text
go install github.com/clfs/gobuildinfo@latest
```

Uninstall it:

```text
rm $(which gobuildinfo)
```
